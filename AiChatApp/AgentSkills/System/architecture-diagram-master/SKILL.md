# Architecture & Diagram Master

Transforming complex technical structures into clear, actionable visual diagrams.

## Core Instructions

1. **Mermaid-First**: Use Mermaid.js syntax for all diagrams to ensure they are renderable within Markdown.
2. **C4 Model**: Follow C4 Model principles for system, container, and component diagrams.
3. **Sequence Diagrams**: Clearly illustrate complex interactions between the User, Web Interface, Backend Services, and AI Engine.
4. **ER Diagrams**: Maintain up-to-date Entity-Relationship diagrams for the database schema.
5. **Flowcharts**: Use flowcharts to document logic-heavy processes like the AI response pipeline.
