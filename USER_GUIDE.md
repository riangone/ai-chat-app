# AiChatApp v2.0 使用与功能手册

欢迎使用 AiChatApp v2.0。本版本引入了全新的 UI 设计、多代理协同模式、长期记忆管理以及技能系统。本文档将引导你充分利用这些新特性。

---

## 🎨 界面与 UI 指南

### 1. 响应式布局 (Responsive Layout)
- **桌面端**：左侧显示会话列表、记忆管理和技能开关；右侧为主要的聊天窗口。
- **移动端/平板**：使用左上角的 **汉堡菜单 (≡)** 唤出侧边栏。点击遮罩层可收回。

### 2. 主题切换 (Theme Switcher)
AiChatApp 内置了 30 多种 DaisyUI 主题（如 Light, Dark, Cupcake, Cyberpunk, Valentine 等）。
- **操作**：点击页面顶部的“Theme”下拉菜单，实时切换全局视觉风格。

### 3. 会话管理
- **新建聊天**：点击侧边栏顶部的 “+ New Chat” 按钮。
- **重命名**：点击会话条目旁边的“铅笔”图标，输入新名称后按回车保存。
- **消息操作**：
    - **复制**：点击消息下方的“Copy”图标。
    - **转发**：点击“Forward”图标，将消息内容填入底部输入框。
    - **存入记忆**：对于 AI 生成的重要信息，点击“Memory”图标将其保存为长期记忆。

## 🤖 核心功能说明

### 1. 多代理协同模式 (Cooperative Multi-Agent)
这是 v2.0 的核心突破，通过三个专门的 AI 角色确保输出质量：
- **Orchestrator (编排者)**：分析任务并制定执行计划。
- **Executor (执行者)**：根据计划生成详细代码或逻辑。
- **Reviewer (评审者)**：校验输出并进行最终润色。
- **如何使用**：在聊天输入框上方的“Cooperative Mode”开关开启即可。适用于复杂的代码编写或系统设计任务。

### 2. 长期记忆 (Long-Term Memory)
AI 不再“健忘”。你可以手动添加或从聊天中保存记忆。
- **机制**：记忆通过“标签 (Tags)”索引。当你的提问中包含某个标签关键词时，相关的记忆会自动注入到 AI 的当前上下文中。
- **管理**：在侧边栏底部的“Long Term Memory”区域查看、添加或删除记忆片段。

### 3. 技能系统 (Skill System)
技能是特定任务的增强插件。
- **示例**：`Python Expert`, `Code Reviewer`, `UI/UX Designer`。
- **如何使用**：在侧边栏“Skills”列表中勾选对应的技能。当技能处于 Active 状态时，AI 会遵循该技能预设的专业指令。

## ⌨️ 快捷操作

- **清空输入**：输入框右侧设有 `✕` 按钮，一键清空当前文本。
- **发送消息**：按 `Enter` 键发送，或点击右侧的发送图标。
- **Markdown 支持**：所有 AI 回复均支持 Markdown 语法（代码高亮、表格、数学公式等），由 `marked.js` 实时渲染。

## 🛠 常见问题 (FAQ)

**Q: 为什么 AI 没有反应？**
A: 请确保本地已安装 `gemini` CLI 且已在 PATH 环境中配置。本应用是该 CLI 的图形化包装。

**Q: 数据库保存在哪里？**
A: 应用使用 SQLite 数据库，文件名为 `chat.db`，位于项目根目录。

**Q: 如何更改 AI 供应商？**
A: 在配置文件 `appsettings.json` 中可以配置不同的 CLI 路径或默认供应商。
